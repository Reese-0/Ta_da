# 获取联通路由器超级管理员账号密码

|          |                     |
| -------- | ------------------- |
| 设备类型 | GPON/4+1+WiFi(2.4G) |
| 设备型号 | G-140W-UD           |
| 硬件版本 | G140WUD_V10         |
| 软件版本 | BEL_UV05_140D       |

1. 192.168.1.1 使用路由器背面账号密码，登录普通账户。
2. 打开设备管理页面
3. F12开发工具查js获得账号密码 通常为CUAdmin CUAdminxxxxxxxx(8位数字)
4. 退出登录管理员账号
